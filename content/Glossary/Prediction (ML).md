> The result produced by an AI system based on patterns in data to identify what output is most likely.

> [!educator] Educators
> 
> Machine learning ML  models are trained to make predictions. The prediction produced by an ML model suggests what the data represents, or what might be useful for a task. For example, an ML developer might train a model to predict which movie someone might want to watch next, based on their viewing habits. The model will generate a prediction after being trained on the movie choices of lots of people. The main job of an ML model is to make predictions. All ML models make predictions, even if in some cases these predictions are not obvious to the user.
