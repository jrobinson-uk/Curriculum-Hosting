>  a common type of lossless compression
> A lossless compression algorithm that involves finding a run of bit patterns, counting the number of occurrences and creating a data pairing using the bit pattern and the number of occurrences. 
> A lossless compression method where repeated runs of binary data are represented by a single instance of the data and the number of repetitions.
