> [!educator] Educators
> 
> Data bias refers to bias reflected in the data used to train machine learning ML  models. Data bias can lead to ML models being trained to generate biased predictions. For example, some facial recognition models are biased against faces of certain skin tones, because the ML models have been trained using mostly images of faces of one skin tone. There are several potential sources of data bias. These include incomplete data, and data that reflects societal bias. Detecting data bias is important to avoid ML models generating biased predictions.
