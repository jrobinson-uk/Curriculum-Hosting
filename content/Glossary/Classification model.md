> [!primary] Younger Learners
> 

> [!secondary] Older Learners
> 
> A type of supervised learning model (see below) that is trained on a large set of data that has been labelled by humans. Human developers will group labelled data into classes. The classification model is trained on labelled data to identify patterns in each class. This allows the model to predict what class new data belongs to. For example, labelling whether a review is positive or negative or identifying animals in a picture.
