> [!primary] Younger Learners
> 
> Part of a program where if a condition is met then a set of commands are run

> [!secondary] Older Learners
> 
>  used when there is more than one possible path for a program to follow
> Controlling the flow of execution in programs using if statements.
> A program control structure that uses a logical expression to determine which path the program follows, depending on whether the expression evaluates to True or False.
